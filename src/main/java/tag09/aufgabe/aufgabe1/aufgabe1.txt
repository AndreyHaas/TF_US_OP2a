Testen einer Methode zur Verarbeitung von Benutzerinformationen
Aufgabenstellung:

1.  Implementiere eine Methode processUser(String username, String password) in einer Klasse UserService, die
    Benutzerinformationen verarbeitet und sicherstellt, dass Benutzername und Passwort nicht leer sind.
    (Das Werfen von IllegalArgumentException nutzen)

2.  Schreibe einen JUnit-Test, um die Methode processUser zu testen. Der Test sollte sicherstellen, dass eine Ausnahme
    geworfen wird, wenn einer der Parameter leer ist.

Hinweis:

Sollte bei der Bearbeitung der Aufgabenstellung Fragen und Unsicherheiten auftreten, besteht die Möglichkeit, eine vereinfachte Variante dieser Aufgabe zu lösen. Dazu wird
die zu prüfende Klasse mit ihren Abhängigkeiten in einer zusätzlichen Textdatei bereitgestellt, die im Ordner/Package "lsg" zu finden ist. In diesem Fall besteht die
Aufgabe nur noch darin, den eigentlichen JUnit-Test zu erstellen.
